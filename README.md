Lien vers l'application Web : https://devweb.iutmetz.univ-lorraine.fr/~e61786u/BUT2/SAE-3.01/SAE301_Finale/src/vue/login.php

1) Créer un compte sur la page register.php

2) Se connecter dessus avec la page login.php

3) Autoriser la géolocalisation / Si indisponible -> Fausse localisation à Metz

4) Guide d'utilisation de l'applicaiton en détails disponible sur le dépot dans le dossier "Documents"

5) Si au premier chargement de index.php rien ne se passe, recharger la page, la map s'affichera

6) Si les parkings de Londres ne s'affichent pas : recharger la page et attendre 5 secondes / si rien : recommencer (environ 3 fois)
